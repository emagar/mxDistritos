From http://alexgaudio.com/2011/07/24/sharing-part-your-repo-github.html

branches: master - all files
          github - subset that will be pushed to gitHub

Add files to subset:

git checkout github
git checkout master name-of-file-to-add
git commit

Drop files from subset:

git checkout github
git rm -rf file-to-drop
git commit

Push to 'mxDistritos' remote repository:
git push mxDistritos github:master
